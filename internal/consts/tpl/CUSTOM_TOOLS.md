# Custom Tools

Document additional tools provided by installed skills or custom integrations here.
This file is loaded into the system prompt alongside the built-in tool definitions.

When you install a new skill that exposes tools, add an entry below so the LLM knows
how to call them. Follow the same format as the built-in tool documentation.
